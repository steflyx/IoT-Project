While completing the project, we had a small doubt on the assignment, which we thought could be interpreted in two different ways:
 - Send a message to the owner of a mote if his mote gets to close to another one
 - Send a message to a controller if two motes get too close to each other
We decided to develop both in respectively, version 1 and version 2.
For version 1, we included a video demonstration on YouTube (available here https://www.youtube.com/watch?v=Cu4VAJeTB1Q).
For version 2, we instead included the more traditional logfile
